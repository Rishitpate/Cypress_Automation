# Cypress_Automation
A simple Cypress demo project with example tests, Cucumber support, and reporting utilities.

## Project Structure

- `cypress/e2e/` - Cypress 15+ component and E2E test files
- `cypress/integration/examples/` - legacy example tests and Cucumber BDD feature files
- `cypress/support/` - custom Cypress commands and E2E configuration
- `package.json` - project dependencies and scripts

## Prerequisites

- Node.js (recommended LTS)
- npm

## Install

```bash
npm install
```

## Run Cypress

```bash
npm run cy:open
```

Then choose a spec to run from the Cypress Test Runner.

## Notes

- Cypress version: `^15.6.0`
- Includes Cucumber preprocessor support via `@badeball/cypress-cucumber-preprocessor`
- Includes reporting and download utilities
